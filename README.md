# ViralNova — Social Media Marketing Agency Website

A premium, fully-responsive marketing agency website built with React, Vite, Tailwind CSS, and Supabase.

## Features

- **Hero** — Bold headline with animated stats counters
- **Services** — 6 service cards with hover effects
- **Before/After Slider** — Interactive drag-to-reveal comparison
- **Portfolio** — 6 case studies with real metrics
- **Testimonials** — 6 client reviews with star ratings
- **Pricing** — 3 tiers (Starter, Growth, Dominator)
- **FAQ** — Smooth CSS accordion
- **Contact Form** — Supabase-backed with validation
- **Floating WhatsApp Button** — Links to +91 6398202864
- **Scroll Reveal Animations** — IntersectionObserver-powered

## Tech Stack

- React 18 + TypeScript
- Vite 5
- Tailwind CSS 3
- Supabase (contact form submissions)
- lucide-react (icons)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
